Landing Page de alta conversão construída puramente com HTML, CSS e Vanilla JS para a venda de curso preparatório para o concurso do IBGE ("Preparação Estratégica"). Sem dependências pesadas, garantindo o máximo de velocidade.

<!-- FEATURES -->
<div align="center">

### &nbsp; 🚀 Funcionalidades

</div>

- **Design Moderno e Responsivo**: Adaptável a dispositivos móveis e desktops, com tema escuro e elementos de "Glassmorphism".
- **Alta Performance**: Zero bibliotecas pesadas (como jQuery) para carregamento instantâneo.
- **Animações e Efeitos**: Transições de entrada suaves e partículas de fundo construídas em Vanilla JS.
- **Upsell & Downsell Funnel**: Inclui páginas otimizadas para ofertas adicionais de alta conversão.
- **Segurança Front-end**: Script leve para proteção básica de cópia e cliques na página.

### &nbsp; 📁 Estrutura de Arquivos

</div>

```
ibge-landing-page/
├── index.html               ← Página principal de vendas
├── style.css                ← Estilos globais desktop
├── mobile-animations.css    ← Animações e responsividade
├── script.js                ← Lógica visual e partículas
├── protection.js            ← Script de proteção
└── upsell/                  ← Funil adicional de ofertas
    ├── index.html           (Página de Upsell)
    ├── downsell.html        (Páginas de Downsell)
    └── obrigado.html        (Agradecimento)
```
